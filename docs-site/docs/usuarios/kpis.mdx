Los KPIs permiten evaluar el impacto y la eficiencia de cada iniciativa.

## Pasos de uso

1. Ingresá a la sección **Mis proyectos**.
2. Consultá los **KPIs** generales o seleccioná un proyecto para indicadores más precisos.
3. Configurá los filtros necesarios para tu análisis.
4. Exportá los resultados si necesitás compartirlos.
