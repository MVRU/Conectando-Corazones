El chat permite coordinar acciones y resolver dudas con rapidez.

## Pasos de uso

1. Abrí un proyecto y seleccioná la pestaña **Chat**.
2. Escribí tu mensaje y enviá para iniciar la conversación.
3. Recibí notificaciones cuando haya respuestas nuevas.
